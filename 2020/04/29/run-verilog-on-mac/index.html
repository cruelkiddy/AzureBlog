主要参考How to use Verilog HDL on Ubuntu 在macOS Mojave10.14.6上实现对Verilog HDL的仿真，并观察信号的波形图。主要依靠Icarus Verilog和GTKWave 两个开源工具

安装

首先安装icarus-verilog brew install icarus-verilog （需要先安装homebrew）

按照这个来安装GTKWave

依照命令行里的提示安装缺失的依赖项（比如一个叫做qu???的软件，和Switch库）。

最后把环境变量加进去，我用的zsh需要在~/.zshrc中添加

export PATH=${PATH}:/Applications/gtkwave.app/Contents/Resources/bin/
